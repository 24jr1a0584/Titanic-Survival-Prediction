# Titanic Survival Prediction

This project predicts whether a passenger survived the Titanic disaster
using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Model
Logistic Regression

## Dataset
Titanic dataset from Kaggle / GitHub.

## Accuracy
Around 80% accuracy.
